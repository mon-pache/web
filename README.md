# Mon‑Paché Web

Página web para el proyecto Mon‑Paché S.L.

## Publicación en GitHub Pages

Repositorio: `mon-pache/web`  
URL esperada: `https://mon-pache.github.io/web/`

## Comandos

```bash
npm install
npm run build
```

Para GitHub Pages, usar como source GitHub Actions o publicar la carpeta `dist`.
